**Percentage & CGPA Calculator**

This is a web-based calculator that helps you calculate **percentage** and **CGPA** by entering marks for your subjects.

It uses basic **HTML, CSS, and JavaScript**, and shows results instantly on the web page.



**💡 What It Does**

- Lets you enter marks for subjects
- Calculates the **percentage**
- Calculates **CGPA** using the standard formula `(Percentage ÷ 9.5)`
- Shows results instantly without refreshing the page



**🛠 Technologies Used**

- **HTML** – Structure of the page  
- **CSS** – Styling and design  
- **JavaScript** – Logic for calculations



**📂 Project Files**

- `index.html` – Main web page  
- `style.css` – Styling  
- `script.js` – Calculation logic


**🚀 How To Use**

1. Open the `index.html` file in a browser (like Chrome or Firefox)  
2. Enter marks in the input boxes  
3. Click “Calculate”  
4. See your **percentage** and **CGPA** immediately
